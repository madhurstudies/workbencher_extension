# codeforces theme

bring colors of codeforces to your workspace.
Preferably can be used in a workspace where you deal with CP.

```json
{
    "workbench.colorCustomizations": {
        "titleBar.activeBackground": "#0067c7",
        "titleBar.activeForeground": "#FFFFFF",
        "activityBar.background": "#c7c400",
        "activityBar.activeBackground": "#9c0000",
        "activityBar.activeBorder": "#0067c7",
        "activityBar.activeForeground": "#FFFFFF",
        "sideBar.background": "#002025",
        "sideBarTitle.foreground": "#FFFFFF",
        "statusBar.background": "#9c0000",
        "statusBar.foreground": "#FFFFFF",
        "tab.activeBackground": "#494400",
        "tab.activeForeground": "#ffffff",
    }
}
````
